## 🛠️ Technical details
* **java.sql** -> The library that manages the connection between the database and the program.
* **DB Browser (SQLite)** -> The application that contains the database.
* **javax.swing** -> The primary Java library used to create the GUIs.
* **Level selecting system** -> The levels are chosen randomly every time you start a new game and are saved into an array.
* **Battle system** -> There is a 50/50 chance that you or the enemy will attack.
